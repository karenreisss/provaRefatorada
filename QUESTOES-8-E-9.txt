-V8
O V8 é um mecanismo de execução de JavaScript de código aberto, desenvolvido pelo Google.
Ele é escrito em C++ e é utilizado tanto no Google Chrome como no Node.js. 
O V8 é responsável por compilar e executar o código JavaScript de forma eficiente, otimizando-o para obter alto desempenho. 
Ele inclui recursos como coleta de lixo (garbage collection), compilação just-in-time (JIT), entre outros, para garantir uma execução rápida e eficiente do código JavaScript. 
Em resumo, o V8 é o coração do ambiente de execução do JavaScript no Chrome e no Node.js, sendo fundamental para o desempenho e a escalabilidade dessas plataformas.
